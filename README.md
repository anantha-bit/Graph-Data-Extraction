# 📊 Graph Data Extraction and Prediction

## 📌 Project Overview

This project extracts data from a graph image and performs trend analysis and future prediction using Linear Regression.

---

## ⚙️ Technologies Used

* Python
* OpenCV
* NumPy
* Pandas
* Matplotlib
* Scikit-learn

---

## 🔍 Workflow

1. Load graph image
2. Apply edge detection
3. Extract data points
4. Remove noise
5. Convert to CSV
6. Apply linear regression
7. Predict future values
